## 🚀 Get Started

You need to be using:

- [Git](https://git-scm.com/downloads)
- [Nodejs v16 or higher](https://nodejs.org/es/download/)

Clone the repository & install dependencies:

```bash
git clone git@github.com:7-18/monitoring-app.git
cd monitoring-app
npm install
```
